# 04_doctor_facility_management.md

## 목표

촉탁의사 DB와 요양원 DB를 관리하는 기능을 구현한다.

---

## 포함 기능

### 촉탁의사 관리

- 목록 조회
- 검색
- 필터
- 신규 등록
- 상세 조회
- 수정
- 상태 변경
- 교육 이수 여부 관리
- 이수증 파일 업로드
- 내부 메모

### 요양원 관리

- 목록 조회
- 검색
- 필터
- 신규 등록
- 상세 조회
- 수정
- 상태 변경
- 내부 메모

## 구현 지시

1. 촉탁의사 CRUD API를 구현한다.
2. 요양원 CRUD API를 구현한다.
3. 목록 테이블 UI를 만든다.
4. 등록/수정 폼을 만든다.
5. 상세 페이지를 만든다.
6. 상태 배지를 만든다.
7. 파일 업로드가 가능하면 이수증 업로드를 구현한다. 어렵다면 파일 메타데이터 구조만 먼저 구현한다.
8. 생성/수정/상태변경 시 AuditLog를 남긴다.

---

## 예외 처리

- 필수값 누락
- 중복 의사 등록 가능성
- 잘못된 연락처 형식
- 존재하지 않는 ID 접근
- 파일 업로드 실패
- 권한 없는 수정
