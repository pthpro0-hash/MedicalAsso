# 03_auth_admin.md

## 목표

관리자 로그인, 권한 구조, 기본 관리자 레이아웃을 구현한다.

---

## 범위

포함:

- 관리자 로그인
- 로그아웃
- 세션 유지
- 권한 기반 메뉴 노출
- 관리자 기본 레이아웃
- 접근 제한

제외:

- 외부 요양원 로그인
- 외부 촉탁의사 로그인
- 소셜 로그인
- 카카오 로그인

---

## 관리자 권한

| 권한 | 설명 |
|---|---|
| SUPER_ADMIN | 전체 설정, 계정 관리, 모든 데이터 접근 |
| STAFF | 일반 사무국 업무 처리 |
| REVIEWER | 추천 후보 검토, 민원 검토 중심 |
| READ_ONLY | 대시보드와 목록 조회만 가능 |

## UI 요구사항

### 로그인 화면

- 서비스명 표시
- 이메일 입력
- 비밀번호 입력
- 로그인 버튼
- 오류 메시지

### 관리자 레이아웃

- 좌측 사이드바
- 상단 사용자 정보
- 로그아웃 버튼
- 현재 메뉴 강조
- 페이지 제목 영역

### 메뉴

```text
대시보드
촉탁의사 관리
요양원 관리
추천 요청 관리
지정/계약 현황
민원·재추천
문서함
설정
```

---

## 보안 기본

1. 비밀번호는 해시 저장
2. 로그인 실패 메시지는 과도한 정보를 주지 않음
3. 인증되지 않은 사용자는 관리자 페이지 접근 불가
4. 권한 없는 사용자는 설정/계정관리 접근 불가
5. 세션 만료 처리

---

## 구현 지시

1. 기존 인증 시스템이 있는지 확인한다.
2. 없으면 관리자용 최소 인증을 구현한다.
3. seed 관리자 계정으로 로그인 가능하게 한다.
4. 권한별 접근 제한을 만든다.
5. 관리자 레이아웃을 구성한다.
6. 이후 기능 페이지가 이 레이아웃 아래 들어가도록 구성한다.
