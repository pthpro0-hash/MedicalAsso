# 촉탁의 추천관리 시스템 제품 방향

## 서비스 정의

이 시스템은 지역의사협의회 사무국이 요양원의 촉탁의사 추천 요청부터 후보 검토, 복수 추천, 최종 지정, 계약 현황, 계약 만료, 민원 및 재추천까지 관리하는 내부 관리자용 행정관리 솔루션이다.

## MVP 포함 범위

- 관리자 로그인 및 권한 관리
- 촉탁의사 후보 DB와 교육 이수 상태 관리
- 요양원 DB 관리
- 추천 요청 접수 및 상태 관리
- 후보 검색, 점수 참고, 복수 후보 추천
- 추천서 PDF 생성 및 문서 이력 관리
- 최종 지정 의사와 계약 기간 관리
- 계약 만료 예정 표시
- 민원 관리 및 재추천 요청 생성
- 대시보드, 업무 큐, AuditLog

## MVP 제외 범위

- 환자별 진료기록, 처방, EMR, 공단 청구
- 요양원 ERP 대체
- 의료 판단 AI와 자동 배정 확정
- 요양원 외부 로그인, 촉탁의사 외부 로그인
- 카카오 로그인
- 알림톡, SMS, 이메일 실발송

## 원칙

진료는 의사가, 운영은 요양원이, 추천 행정은 의사회가 한다. 이 시스템은 의료행위 관리 시스템이 아니며, 의사회의 추천 행정을 정확하고 빠르게 처리하기 위한 관리자용 업무 도구다.

## 상태 흐름

추천 요청은 `RECEIVED -> UNDER_REVIEW -> CANDIDATES_SELECTED -> SENT_TO_FACILITY -> ASSIGNED -> CONTRACT_REGISTERED -> CLOSED` 흐름을 기본으로 한다. 예외 상태는 `CANCELED`이며, 주요 상태 변경은 AuditLog에 기록한다.

## 용어

| 용어 | 의미 |
|---|---|
| 촉탁의사 | 요양원에 정기적으로 방문하거나 계약되는 의사 후보 |
| 요양원 | 촉탁의 추천을 요청하는 장기요양시설 |
| 추천 요청 | 요양원이 촉탁의 후보 추천을 요청한 업무 건 |
| 복수 추천 | 의사회가 여러 명의 후보 의사를 추천하는 행위 |
| 지정 결과 | 요양원이 추천 후보 중 최종 선택한 의사 |
| 계약 | 지정된 촉탁의사와 요양원의 운영 기간 |
| 민원 | 추천, 계약, 일정, 소통 관련 행정 이슈 |
| 재추천 | 기존 지정 또는 계약 이슈로 다시 후보를 추천하는 업무 |
